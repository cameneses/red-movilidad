## Getting Started

### Usage

1. Clone the repo
   ```sh
   git clone https://github.com/cameneses/red-movilidad
   ```
2. Install packages
   ```sh
   yarn install
   ```
3. Run the server
   ```sh
   yarn run dev
   ```
